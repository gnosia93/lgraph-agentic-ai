100B급 대형 멀티모달 모델(VLM)을 교사(Teacher)로 삼아 1B급 소형 정제 모델(Student)을 만드는 과정은 AI 인프라와 데이터 엔지니어링의 정수가 필요한 '지식 증류(Knowledge Distillation)' 및 '합성 데이터 생성(Synthetic Data Generation)' 프로젝트입니다.
특히 멀티모달(텍스트+이미지 등) 영역은 단순 텍스트보다 정렬(Alignment)이 까다롭기 때문에 전략적인 접근이 필요합니다. 

### 100B -> 1B 멀티모달 지식 증류 아키텍처 ###

### 1단계: 교사 모델(100B VLM)을 통한 데이터 정제 및 합성 ###
1B 모델은 파라미터가 작기 때문에 지저분한 데이터를 학습하면 성능이 급격히 망가집니다. 100B 모델을 일종의 **'데이터 공장 및 정제소'**로 활용해야 합니다.

* 고품질 캡셔닝 및 QA 세트 생성 (시각적 지식 증류)
  *	대규모 이미지 데이터셋(예: LAION 등)을 준비한 뒤, 100B VLM에게 각 이미지에 대한 극도로 상세한 설명(Dense Captioning)과 복잡한 추론 질문/답변 쌍(VQA)을 생성하게 합니다.
  *	프롬프트 엔지니어링: "생각의 흐름(CoT, Chain-of-Thought)을 포함하여 이 이미지의 인과관계를 단계별로 설명해줘" 형태로 100B의 추론 프로세스 자체를 데이터에 녹여냅니다.
* 데이터 필터링 (품질 검증)
  * 100B 모델이 뱉은 합성 데이터 중 리워드 모델(Reward Model)이나 독성/환각 스캔을 거쳐 품질 점수가 높은 상위 데이터만 정제하여 1B 모델의 학습셋으로 구축합니다.


### 2단계: 소형 모델(1B) 아키텍처 및 정렬(Alignment) 설계 ###
1B 모델은 이미지와 텍스트를 모두 이해할 수 있는 경량 아키텍처로 세팅해야 합니다. 최근 트렌드는 LLaVA 스타일이나 SigLIP + 경량 LLM 컴포넌트 조합입니다.
* 비전 인코더 활용: 이미지 특징 추출은 처음부터 학습시키지 않고, 이미 검증된 오픈소스 비전 인코더(예: SigLIP-SO400M 또는 CLIP-ViT-L)를 고정(Freeze)하여 사용합니다.
* 프로젝션 레이어(연결부) 훈련: 비전 인코더가 뽑은 이미지 벡터를 1B 언어 모델이 이해할 수 있는 차원(Dimension)으로 변환해 주는 가벼운 MLP(Multi-Layer Perceptron) 레이어를 설계합니다.
* 언어 모델(Backbone): 대형 모델에서 텍스트 지식이 잘 전수될 수 있도록 Llama-3-1B나 Qwen2.5-1.5B 같은 검증된 경량 백본을 기반으로 시작합니다.

### 3단계: 인프라 기반 분산 학습 전략 ###
1B 모델 학습이지만 멀티모달 데이터(고해상도 이미지 텐서)는 메모리를 엄청나게 잡아먹기 때문에 다음과 같이 학습을 분기해야 합니다.

#### ① 1단계: 프로젝션 정렬 학습 (Freeze LLM) ####
•	목표: 이미지와 텍스트 벡터 공간 맞추기
•	인프라: 1B 모델 자체는 작으므로 PyTorch Native FSDP를 사용해 싱글 노드 멀티 GPU 환경(training-on-eks 내 L40S 혹은 A100 클러스터)에서 빠르게 끝낼 수 있습니다. 이 단계에서는 LLM 가중치는 잠그고 연결 부위만 학습시킵니다.
