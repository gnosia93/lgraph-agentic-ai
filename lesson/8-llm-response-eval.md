LLM 답변 품질 평가는 크게 자동 평가와 사람 평가로 나뉘고, 실무에서는 둘을 섞어서 씁니다. 대표적인 방식들을 정리해드릴게요.

기준(무엇을 평가하나) 먼저 "품질"을 여러 축으로 나눕니다.

* 정확성(Accuracy / Factuality): 사실이 맞는지, 환각(hallucination)이 없는지
* 관련성(Relevance): 질문 의도에 맞는 답인지
* 완결성(Completeness): 빠진 내용 없이 충분한지
* 일관성/논리성(Coherence): 앞뒤가 맞고 논리적인지
* 안전성(Safety): 유해·편향·민감 정보 문제가 없는지
* 형식 준수(Instruction following): 요청한 형식·길이·톤을 지켰는지
